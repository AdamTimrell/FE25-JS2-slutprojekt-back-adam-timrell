SLUTPROJEKT JS2 - Scrum Board           

Detta är ett scrum board-projekt med. 
- Backend: Node-js + Express + TypeScript
- Frontend: Vite +  TypeScript + Bootstrap
- Datan lagras i src/data/db.json

krav för att köra
- Node.js installerat
- npm installerat

För att starta Projektet

1. Starta backend
Öppna terminal i projektroten (backend-slutprojekt) och kör: 

npm install
npm run dev

Backend kör på: http://localhost:3000

2. Starta Frontend 
Öppna en ny terminal i Front-end slutprojekt och kör: 

npm install
npm run dev

Frontend kör på: http://localhost:5173 eller den URL som vite visar i terminalen
